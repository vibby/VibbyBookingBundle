VibbyBoookingBundle
===================

Sf2 ressources-booking system Bundle

Installation
===================

 
 curl -s https://getcomposer.org/installer | php
 
 vim composer.json
 
paste this :

 {
   "minimum-stability": "dev",
   "require": {
      "vibby/vibby-booking-bundle": "*"
   },
   "repositories": [
     {
          "type": "git",
          "url": "https://github.com/vibby/VibbyBookingBundle.git"
     }
   ]
 }
